# botnet-project
Botnet create for school project

Requirements:

	pynput.keyboard
	pyfiglet
	termcolor
	
How to run:

	python3 Master.py -H <IP> -P <PORT> 
	
by default the IP is localhost and the port is 6666 (same as slave port)
